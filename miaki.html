How to install Miaki?

What do you need?

First you need enso/enso_ex (enso_ex v5.X recommended)

You need a DevKit PUP It is important to note that you need a PUP Devkit of the same version as your current firmware Download here!

Then you need to install the VPK with VitaShell

Before installation

Go to Safe mode and do Rebuild database

Download the CEX PUP if you're on a retail version or DEX if you're on a testkit of the same or higher firmware that you currently have and put it on your official memory card (not the SD2VITA) and put it in ux0 (or uma0): data/PSP2/UPDATE/PSP2UPDAT.PUP like that if you have a soft-brick with miaki you can always restore in safe mode by updating with the memory card

Installation

Open Miaki then do "Install TOOL Firmware" then I advise you to do e expired TOOL to the question (henkaku works badly with the activated kit) then you have to wait 20-45 minutes for the installation and then it's good! you have a firmware TOOL

What is RTU/ROOL/REX?

RTU: Retail-Internal Test Unit

ROOL: Retail-DevTool

REX: Retail-Debug

Features

What working?

DevMode

Activate/Deactivate DEX spoof

ProductCode Switch (RTU/ROOL/REX)

APP_HOME, which works with ux0, sd0, and host0 (usbhostfs)

Activation REX spoof

What doesn't work?

Neighborhood because there is no Communication Processor (CP)

Activation in ROOL spoof and RTU

When you have the spoof tool in the bottom left corner, you'll see a message saying "The backup battery has failed."

And in the spoof RTU, the PS Vita is activated unlimitedly.

Problem

In DevMode, the debug settings System Update, Network, PSN, System, and Libgxm will crash the console (Registry issue). Otherwise, in Release Mode, everything works.

How to fix some issues

1. Debug settings in DevMode - you can use VshPatch or APP_HOME and replace Mini Settings for QA with the eboot.bin from the testkit version. You will no longer have crashes, but some DevMode settings will be missing (LIBGXM).
2. Neighborhood - No.
3. Activation in spoof TOOL - you can use Red-msg-delete to remove the red message.
